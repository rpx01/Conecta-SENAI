Perguntas Frequentes

Para alterar o módulo padrão que abre ao iniciar o sistema, ative a opção "Lembrar minha escolha" na tela de seleção de sistemas e entre no módulo desejado. O sistema salvará essa preferência automaticamente para os próximos acessos.

O acesso aos módulos é gerenciado pelo administrador do sistema. Você deve abrir um chamado no módulo Suporte de TI solicitando a liberação de acesso para o módulo desejado, informando a justificativa.

Os relatórios de ocupação estão disponíveis no módulo Controle de Ocupação de Salas. Acesse o Dashboard e utilize os botões de exportação (PDF, Excel, CSV) localizados nos cards de estatísticas.

Agenda de Laboratórios

Como usar o Módulo

O módulo de Agenda de Laboratórios permite gerenciar reservas de espaços, visualizar disponibilidade em tempo real e controlar o uso dos laboratórios do SENAI.

Principais Funcionalidades:
  • Visualização em Calendário: Veja todos osagendamentos em uma interface intuitiva
  • Novo Agendamento: Reserve laboratórios com poucos cliques
  • Gerenciamento de Turmas: Associe turmas aos agendamentos
  • Controle de Salas: Visualize status e disponibilidade
Primeiros Passos:
  1. Acesse o módulo pela tela de seleção de sistemas
  2. Visualize o calendário com agendamentos existentes
  3. Clique em "Novo Agendamento" para criar uma reserva
  4. Preencha os campos obrigatórios e confirme

Agenda de Treinamentos

Como usar o Módulo

O módulo de Agenda de Treinamentos facilita a programação e gestão de cursos técnicos, permitindo controle completo sobre inscrições, turmas e histórico.

Principais Funcionalidades:
  • Catálogo de Cursos: Visualize e gerencie todos os cursos disponíveis
  • Gestão de Turmas: Crie e administre turmas de treinamento
  • Controle de Inscrições: Gerencie inscrições de participantes
  • Histórico de Treinamentos: Acesse registros de cursos anteriores
Dica Importante:

Use os filtros de data e área para localizar rapidamente os treinamentos desejados.

Controle de Ocupação de Salas

Como usar o Módulo

O módulo de Controle de Ocupação monitora e registra o uso das salas de aula, gerando estatísticas de utilização para otimizar a gestão de espaços.

Principais Funcionalidades:
  • Dashboard de Ocupação: Visualize estatísticas em tempo real
  • Registro de Ocupações: Controle entrada e saída de turmas
  • Relatórios de Utilização: Gere relatórios em PDF, Excel ou CSV
  • Base de Dados: Gerencie salas, turmas e corpo docente
Como Exportar Relatórios:
  1. Acesse o Dashboard de Ocupação
  2. Utilize os filtros de data para selecionar o período
  3. Clique nos botões de exportação (PDF/Excel/CSV) nos cards

Portal de Notícias

Como usar o Módulo

O Portal de Notícias centraliza comunicados institucionais, destaques e aniversariantes, mantendo todos informados sobre eventos e novidades do SENAI.

Principais Funcionalidades:
  • Visualização de Notícias: Acesse comunicados importantes
  • Filtros por Categoria: Encontre notícias por tipo ou data
  • Aniversariantes do Mês: Veja quem está de aniversário
  • Gerenciamento (Admin): Publique e edite notícias
Para Administradores:

Para publicar uma notícia, acesse "Gerenciar" no menu lateral, clique em "Nova Notícia" e preencha os campos de título, conteúdo e categoria.

Suporte de TI

Como usar o Módulo

O módulo de Suporte de TI permite abrir chamados técnicos e acompanhar solicitações de suporte ao parque de TI do SENAI.

Principais Funcionalidades:
  • Abertura de Chamados: Registre problemas técnicos
  • Acompanhamento: Veja status dos seus chamados
  • Histórico: Consulte chamados anteriores
  • Indicadores (Admin): Visualize métricas de atendimento
Como Abrir um Chamado:
  1. Clique em "Abrir Chamado"
  2. Selecione a categoria do problema
  3. Descreva detalhadamente o ocorrido
  4. Adicione capturas de tela se relevante
  5. Clique em "Enviar Chamado"

Dica: Antes de abrir um chamado, consulte o FAQ acima - sua dúvida pode já estar respondida!

Controle de Rateio

Como usar o Módulo

O módulo de Controle de Rateio permite a apuração de horas de instrutores e rateio de custos entre diferentes áreas e projetos.

Principais Funcionalidades:
  • Dashboard Financeiro: Visualize métricas de custos e alocações
  • Apuração de Horas: Registre e calcule horas trabalhadas
  • Relatórios de Rateio: Gere relatórios detalhados por período
  • Gestão de Instrutores: Controle cadastro e dados de instrutores
Glossário de Termos:
Rateio:
Distribuição proporcional de custos entre diferentes centros de custo.
Apuração de Horas:
Processo de contabilização das horas trabalhadas por instrutores.
Centro de Custo:
Unidade organizacional à qual os custos são alocados.

Gerenciamento de Usuários

Como usar o Módulo

O módulo de Gerenciamento de Usuários permite cadastrar, editar e controlar permissões de acesso ao sistema Conecta SENAI. Acesso restrito a administradores.

Principais Funcionalidades:
  • Cadastro de Usuários: Crie novos usuários no sistema
  • Controle de Permissões: Defina quais módulos cada usuário pode acessar
  • Edição de Perfis: Atualize dados e permissões
  • Inativação/Reativação: Gerencie status de usuários
Níveis de Permissão:
  • Administrador: Acesso completo a todos os módulos
  • Gestor: Acesso a módulos operacionais e relatórios
  • Usuário Padrão: Acesso básico aos módulos permitidos

Importante: Alterações em permissões têm efeito imediato. Sempre confirme as mudanças antes de salvar.
